# Diploma
В работе реализована модель распространения информации в сети с учетом:
- типа информации;
- типа источника;
- уровня доверия к источнику.

Также реализованы:
- генерация сети;
- моделирование каскадного распространения;
- случайный метод выбора начальных узлов;
- алгоритм CELF;
- TIM-подобный метод.
